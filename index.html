Useless Machine

Useless cover version of an eye chart. This generates a new experience by encouraging viewers to search for hidden element (E facing on the other side) to only see none as it is actually a completely useless eye test.

Escape

An exploration of how the visual can embed meaning into the word's graphic representation. 'Escape' can be defined as 'fail to be noticed.' Using this aspect of the word, a 30"x30" puzzle was created for viewers to find the word 'escape' which exclusively existed among the letters E, S, C, A and P.

Hair Loss Statistic Calendar

"Hair loss sufferers lose approximately 100 strands each day" Using calendar as a format to convey this statistic by visually demonstrating hair loss progress. The man loses more hair as the day goes by.

"THANKS, YOUR ENGLISH IS GOOD TOO!"

Designing a poster for a window facing the South Main Street, addressing the world’s current issue or share a message. This poster presents a response to “Your English is so good!,” compliment a lot of Asian often get from Americans. This poster may seem funny when you first read it, but it is made as a sarcasm.
